# Google Go
https://golang.org/

## Downloads
https://golang.org/dl/

## Install for Windows

* ファイルをダウンロード　[go1.12.2.windows-amd64.zip](https://dl.google.com/go/go1.12.2.windows-amd64.zip)
* ファイルを展開する。例) C:/Go
* 環境パス設定 GOROOT=c:/Go  PATH=%GOROOT%/bin;%PATH%

## バージョン確認
```
$ go version
go version go1.12.2 windows/amd64
```
